# Лабораторна робота №5. Робота з системою контролю версій Git

**Тема:** Розгалужена робота у системі контролю версій Git. Робота з віддаленими репозиторіями.

**Опис:** Ця лабораторна робота присвячена освоєнню продвинутих механіків Git, таких як створення гілок, їх злиття, вирішення конфліктів та робота з віддаленими репозиторіями.

**Мета:** Опанувати практичні навички розгалуженої роботи в Git, навчитися створювати, об'єднувати гілки та взаємодіяти з віддаленим репозиторієм.
